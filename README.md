## ❗ Info
Used to get runnable lua code from dumped luac files.

## ❓Usage
### 1.)
=strg c + strg v your luac into the dir and rename it to input.luac

### 2.)
#=run decompile.bat

### 3.)
Your lua code should now be in the "output.lua" file
